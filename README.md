# Bright Future Primary School Student Portal

A modern Student Management System built with Java Spring Boot and Thymeleaf.

## 🚀 Features
- Student Registration with photo upload
- Complete CRUD operations
- Admin Dashboard with statistics
- Search and pagination
- Responsive design

## 🛠️ Tech Stack
- Java 21, Spring Boot 3.2
- Thymeleaf, Bootstrap 5
- MySQL, Spring Data JPA
- Spring Security

## ⚙️ Setup
1. Create database: `school_portal`
2. Update `application.properties` with MySQL password
3. Run: `mvn spring-boot:run`
4. Access: http://localhost:8080
